# debianX11termux
single script to install and setting up debian in termux for x11

1. Clone this repo /n
2. '''chmod +x ./setup-debian-x11.sh''' /n
3. Start script '''./setup-debian-x11.sh''' /n

/n
To start X11 run new script /n
'''./start-xfce.sh''' /n
To stop just run script /n
'''./stop-xfce.sh'''
